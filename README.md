# Heart Disease Analysis App

## Final Project — Tecnolochicas: Introduction to Data Science

This project is the final project for the **Tecnolochicas Introduction to Data Science** program.  
It explores clinical data related to heart disease and presents key insights through a mobile app built with **MIT App Inventor**.

---

## Dataset

Data source:  
https://www.kaggle.com/datasets/neurocipher/heartdisease

From the original dataset, this project uses:
- **Age**
- **Cholesterol**
- **Heart Disease** (target variable)

The **Heart Disease** column was converted into a **binary system**:
- `1` → Presence of heart disease  
- `0` → No heart disease

## App Description

The app connects to a Google Sheets database and displays selected variables to help users understand the relationship between age, cholesterol, and heart disease.

---

## Purpose

This project demonstrates basic data cleaning, feature selection, and binary classification concepts applied to health data.
